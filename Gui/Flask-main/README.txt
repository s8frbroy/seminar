OVERVIEW:

-- Folders
* appendix:  not relevant for the GUI (has an extra README file)
* model:     stores our final (LSTM and CNN) models which will be used for prediction
* static:    is a "helper" folder for the GUI which stores uploaded and intermediate data files
* templates: includes all html files for the GUI

-- Python files
* main:      run this file in the console of your choice
* predict:   used for prediction needed in main file
* util:	     helper file (most functions are used for preprocessing)
